## What is `Caddy`

`Caddy` is an open-source [web server](./web-infrastructure.md#web-server) and [reverse proxy](./web-infrastructure.md#reverse-proxy).

`Caddy` is configured via a [`Caddyfile`](#caddyfile)

Docs:

- [Caddy](https://caddyserver.com/docs/)

### `Caddyfile`

A `Caddyfile` is [`Caddy`](#caddy)'s configuration file. It defines which port `Caddy` [listens](./computer-networks.md#listen-on-a-port) on and where to [forward requests](./web-infrastructure.md#forward-request).

Docs:

- [Caddyfile concepts](https://caddyserver.com/docs/caddyfile/concepts)
- [Environment variables in `Caddyfile`](https://caddyserver.com/docs/caddyfile/concepts#environment-variables)
