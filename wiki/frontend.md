## What is frontend

A frontend is a client-side [application](./software-types.md#application) that runs in a [browser](./useful-programs.md#browser) and provides the visual interface that users interact with.

It communicates with a [backend](./backend.md#what-is-a-backend) through a [web API](./web-api.md#what-is-a-web-api) to display and manipulate data.

A frontend is typically built with `HTML`, `CSS`, and `JavaScript`.

Modern frontends use frameworks like `React`, `Vue`, or `Angular` to organize the interface into reusable components.
