# `Autochecker` API

## About the `Autochecker` API

The `Autochecker` API lets you query the anonymized data collected by the [`Autochecker`](./autochecker.md#what-is-the-autochecker).

There are several [`Autochecker` API endpoints](#autochecker-api-endpoints) accessible after authentication using your[`Autochecker` API credentials](#autochecker-api-credentials).

## `Autochecker` API credentials

- [Login](#autochecker-api-login)
- [Password](#autochecker-api-password)

### `Autochecker` API login

Your University email.

Example: `j.doe@innopolis.university`.
